# VIZ V2 — Live HUD + NOW 한 줄 + LLM 활동 요약

V1 에서 두 가지 추가:

## V1 → V2 변경점

### 1. NOW 한 줄 (헤더 아래 큰 박스)
지금 이 순간 Claude가 무엇을 하고 있는지 자연어 한 줄로 큰 글씨로.

```
┌────────────────────────────────────────┐
│ 지금                                    │
│ ⚡ 명령 실행 중                          │
│ pytest tests/ -v --tb=short             │
└────────────────────────────────────────┘
```

- 도구 호출이 올 때마다 갱신
- 8초 idle 후 "대기 중"으로 전환
- LLM 호출 없음 (규칙 기반, 즉시)

### 2. LLM 활동 요약 (Stop 이벤트마다)
Claude 응답 끝날 때마다 그 turn의 활동을 한국어 자연어로 압축한 카드.

```
┌─ 활동 요약 ────────────────────────────┐
│ 📝 [LLM] stop · 12 events              │
│ api.py에서 데이터베이스 연결 함수 두    │
│ 곳 수정, pytest로 단위 테스트 검증.     │
│ → 통합 테스트 추가 예정                 │
│ [api.py] [tests/test_api.py]            │
└────────────────────────────────────────┘
```

- 트리거: Stop / 30개 이벤트 누적 / 60초마다 강제
- 사용 모델: claude-haiku-4-5 (빠르고 싸다)
- API key 없으면 규칙 기반 fallback (도구 카운트만)

## LLM 활성화

```bash
export ANTHROPIC_API_KEY='sk-ant-...'
./start.sh
```

브라우저 헤더 우상단 뱃지 확인:
- 초록 `LLM ON` = 활성
- 노랑 `LLM OFF` = 비활성 (fallback 사용)

## 비용 (대략)
- claude-haiku-4-5 가격: 입력 $0.25/M 토큰, 출력 $1.25/M 토큰
- 요약 1건당 ~600 in / ~200 out 토큰 ≈ **$0.0004 미만**
- 하루 100건 요약해도 약 4센트

## V2의 한계 (V3 후보)
- 요약은 turn 단위 — 더 긴 호흡(시간/feature 단위) 압축 없음
- NOW 가 항상 영어 (도구 이름 그대로) — 더 자연어로 가능
- 카드 간 연결선 없음
- 파일 트리/그래프 뷰 없음
- 코드 diff 시각화 없음
