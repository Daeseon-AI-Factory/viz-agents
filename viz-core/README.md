# 🎯 viz-core

**나의 viz 지식 저장소.** 본인이 이해한 개념과 시스템을 시각으로 영구 저장 + AI 출력 시각화 레이어.

`AI 출력 → viz-core → 사용자의 0.5초 글랜스`
`나의 학습 → library → 다시 떠올릴 때 즉시`

자세한 정의는 [../CORE.md](../CORE.md) 참고.

---

## 핵심 페이지

| 경로 | 무엇 |
|---|---|
| `/` | 메인 — AI 출력 실시간 시각화 |
| `/library.html` | 📚 **내 저장소** — 모든 24종 viz_kind 자산 통합 |
| `/concepts.html` | 💡 병신교육소 — 기술 개념 비유 시각화 |
| `/showcase` | 24종 viz_kind 샘플 갤러리 |
| `/spec` | AI 가이드 — viz-spec 마커 명세 |

## library/ 폴더 (영구 자산)

```
library/{viz_kind}/{slug}.json
├── concept/        ← 기술 개념 (캐싱/Index/JWT...)
├── depgraph/       ← 본인 시스템 의존도
├── arch/           ← 본인 아키텍처
└── ... (24종 가능)
```

- 추가: 폼 또는 JSON 직접 편집 (`/library.html`)
- 백업: `/library/export` (zip 다운로드)
- 복원: `/library/import` (zip 또는 JSON 업로드)

## 실행

```bash
./start.sh
```

자동:
- `.venv` 생성 + 의존성 설치
- 이전 v* 폴더의 키 자동 복사
- hooks 설치 (이미 있으면 skip)
- 브라우저 자동 열림

## 핵심 기능 (10개)

1. **NOW 박스** — 지금 뭐 하는지 한 줄
2. **세션 박스 + ▼ 흐름** — 여러 세션 색별 통합
3. **LLM 활동 요약** — Stop 시 한국어 한 줄
4. **11종 viz_kind 자동 매핑** — diff/gauge/table/flow/badge/code/animation/kpi/journey/arch/whatif
5. **AskUserQuestion 옵션 카드** — 자동 그리드
6. **자연어 시각 요청** — 입력 박스 → LLM → 시각
7. **우클릭 → 클립보드** — Claude 채팅에 컨텍스트 다리
8. **⭐ 즐겨찾기** — 마음에 든 시각 저장
9. **⚙️ 키 모달** — 디스크 저장 + 자동 로드
10. **다크/라이트 자동** — OS 설정 따라감

## Endpoints (7개)

| | |
|---|---|
| `GET  /` | index.html |
| `GET  /healthz` | 상태 |
| `WS   /ws` | 실시간 push |
| `POST /event` | Hook 이벤트 수신 |
| `GET  /history?limit=N` | 과거 이벤트 |
| `POST /viz/request` | 자연어 요청 |
| `POST /favorite` | ⭐ 저장 |
| `GET/POST /key*` | 키 관리 |
