viz-core Showcase

모든 viz_kind 의 샘플. 본인이 원하는 수준 평가용. 박이 안 좋으면 본인이 어떤 게 좋은지 알려주면 fix.

gauge단일 수치 / 진행률
{"label":"테스트 통과율","value":12,"max":15,"unit":"통과"}

{"label":"통과","tone":"success"} / error / warning / info

{"kpis":[{"label":"MRR","value":"$52K","trend":"up","delta":"+2.3%"},...]}

📊 Phase 1 — 차트 (Chart.js)

timeseries ★시계열 추세 — 매출/사용자/응답시간
{"labels":["Jan",...],"series":[{"label":"매출","values":[...]}]}
bar ★카테고리 비교 — 지역/팀/제품별
{"labels":["서울","부산","대구",...],"values":[...]}
funnel ★전환 깔때기 — 방문→가입→결제
{"stages":[{"name":"방문","value":10000},{"name":"가입","value":3200},...]}

⚙️ Phase 2 — 운영/프로젝트

heatmap ★시간×지표 분포 — CPU/메모리/트래픽
{"labels_x":["0시","4시",...],"labels_y":["월","화",...],"matrix":[[...]]}
kanban ★작업 보드 — TODO/DOING/DONE
{"columns":[{"name":"TODO","cards":[{"title":"...","assignee":"...","tags":[...]}]},...]}

📈 Phase 3 — 고급 분석

waterfall ★분산 추적 — API trace span 시간축
{"spans":[{"name":"API /users","service":"gateway","start_ms":0,"duration_ms":120}]}
cohort ★retention 매트릭스 — 월별 유지율
{"cohorts":[{"label":"2026-01","initial":1000,"retention":[1000,650,420,300,...]}]}

🎨 Phase 4 — PM/UX/시스템 설계

crud ★권한 매트릭스 — 엔티티 × 역할 (PM/설계 필수)
{"entities":["User","Order"],"actors":["User","Admin"],"matrix":[[["R","U"],["C","R","U","D"]],...]}
userflow ★사용자 동선 + 분기 (결정 노드)
{"nodes":[{"id":"visit","kind":"start"},{"id":"loggedin","kind":"decision"}],"edges":[{"from":"visit","to":"loggedin"}]}
screenmap ★화면 박스 + 전환 화살표 (UX 와이어프레임)
{"screens":[{"id":"home","title":"홈","x":0,"y":0,"items":[...]}],"transitions":[{"from":"home","to":"detail","action":"클릭"}]}
depgraph ★서비스 의존성 그래프 (마이크로서비스)
{"services":[{"id":"gw","label":"Gateway","kind":"frontend"}],"deps":[{"from":"gw","to":"auth","kind":"http"}]}

💡 Phase 5 — 기술 개념 이해 (병신도 이해)

AI 가 추천하는 기술(캐싱/Index/JWT 등) — buzzword 빼고 비유 + 시각 + 장단 + 실무로 즉시 이해.

concept ★★캐싱 — 자주 쓰는 정보는 가까이
{"concept":"캐싱","tagline":"...","analogy":{"name":"책상 위 메모지","icon":"📝"},"comparison":{...},"tradeoffs":{...},"real_world":"..."}
concept ★★Index — 백과사전 색인
concept ★★JWT — 영화관 티켓
