# VIZ V3 — Live HUD + 세션별 흐름 그래프

V2 의 **카드 timeline**을 **세션별 박스 + 노드 흐름 그래프**로 재배치.

## V2 → V3 변경점

### 변화
```
V2 (역시간 카드 timeline)            V3 (세션 박스 + 흐름 그래프)
                                     
[최신]   📝 활동 요약                 ┌─ Session A (분홍) ──────┐
         ⚡ Bash pytest               │ 💬 사용자                │
         ✏️ Edit api.py               │  ▼                      │
         🔍 Grep                      │ 📖 Read api.py          │
         📖 Read                      │  ▼                      │
         💬 사용자                    │ 🔍 Grep verify_pwd      │
[과거]   (옛 세션도 위에 끼어있음)    │  ▼                      │
                                     │ ✏️ Edit api.py          │
                                     │  ▼                      │
                                     │ ⚡ pytest ✓             │
                                     │  ▼                      │
                                     │ 📝 활동 요약             │
                                     └─────────────────────────┘
                                     
                                     ┌─ Session B (파랑) ──────┐
                                     │ ...                     │
                                     └─────────────────────────┘
```

### 박스 정렬 로직
- 새 활동 발생한 세션이 **위로** 올라감 (자연 활성 → 자연스레 시야 안)
- 각 박스 안에서는 **위→아래 시간 흐름** + 노드 사이 `▼` 화살표
- 세션마다 다른 색 (좌측 strip + dot)

### 같음 (V2와)
- 포트 8765, hooks 태그 동일
- 키 모달 (⚙️ 키 설정 버튼)
- NOW 박스
- LLM 활동 요약 (Stop마다, 4 events 이상)
- `.local_key.txt` 디스크 키 저장

## 사용

```bash
cd v3
./start.sh
```

V2의 키를 자동으로 V3로 복사 (있으면). 처음이면 ⚙️ 키 설정에서 한 번만.

- 박스 안 노드 간 연결선은 단순 `▼` (SVG/곡선 아님)
- 분기/병합 표현 없음 (서브에이전트 spawn 같은 거)
- 코드 diff 시각화 없음 — V4 후보
- 파일 트리 사이드패널 없음 — V4 후보
